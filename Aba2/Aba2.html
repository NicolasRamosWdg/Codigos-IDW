CSS e suas Variações

Todos nós sabemos, que para dar um "chan" a amis no nosso site, nós devemos usar o máximo da criatividade, e com ela, deixar o nosso site maios bonito. Para isso usamos o CSS, mas existe o CSS Inline, interno e externo. Quais as Diferenças?

INLINE

Usado para dar "estilo" HTML específico. Para usae ele, basta adicionar o atributo "style" para cada TAG HTML sem usar os seletores

Estetipo de CSS não é muito prátio, pois par usar ele, nós devemos ultilizar de maneira individual.

Contudo, o CSS nline no HTML pode ser útil para algumas situações. Por exemplo, sem casos onde você não tem acesso aos arquivos CSS ou precisa aplicar estilos para um elemento único.

EXTERNO

Este tipo de CSS é um método mais eficiente, especialmente se você está estilizando um site grande. Ao editar um arquivo .css, você pode modificar um site inteiro de uma só vez. Com o CSS externo, você vai linkar as páginas da internet com um arquivo .css

INTERNO

O CSS interno ou incorporado requer que você adicione a tags "style" na seção "head" do seu documento HTML.

Este estilo de CSS é um método efetivo de estilizar uma única página. Contudo, usar esse estilo em múltiplas páginas pode consumir muito tempo, já que você precisa definir as regras CSS para cada página do seu site.

Diferenças entre os seletores CSS: Classe, Elemento e ID.

  1. ID: IDs são identificadores, e por isso, não devem se repetir em vários elementos. Em uma sessão, você cria um ID, nele você configura cor do texto, tamanho, fonte etc. E quando você quiser, você chama essa ID.
  2. Elemento: São os que constituem a base de todo documento HTML.
  3. Classe: Permitem o CSS selecionar e acessar elementos específicos através dos seletores de classe.

5 propriedades CSS

  1. style: Usada para estilizar qualque item da página.
    Exemplo: "style="color:Tomato;">Hello World"
  2. background-color: Propriedade específica para por cor no fundo de um elemento.
    Exemplo: "background-color: lightblue;"
  3. border: Usada para colocar borda na estrutura HTML.
    Exemplo: "border-style: dotted;"
  4. align: Usada para definir a largura do elemento impedirá que ele se estenda até as bordas.
    Exemplo: "margin: auto;"
  5. float: A propriedade float é usada para posicionar e formatar conteúdo, por exemplo, deixe uma imagem flutuar à esquerda do texto.
    Exemplo: "float: right;"
